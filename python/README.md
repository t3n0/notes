# 🐍 Python Notes

Collections of notes that I built over the years with the most various purposes:

- [**miniconda installation**](notes/installation.md) - Newbie tutorial for miniconda installation plus the most basic commands.
- [**build, distribute and install**](notes/python-packaging.md) - Step by step guide on how to build and distribute a python package.
- [**extension modules with cython**](notes/cython.md) - Automatizing the compilation and building of extension modules written in cython.
- [**multithreading**](notes/multithreading.md) - Basic multithreading tutorial for python.
- [**multiprocessing**](notes/multiprocessing.md) - Simple script to implement a multiprocessing python code.
- [**MPI with python**](notes/mpi4py.md) - How to use `mpi4py` in a very simple example.
- [**system-wide scripts**](notes/python-scripts.md) - Save and run a script all over your machine.
